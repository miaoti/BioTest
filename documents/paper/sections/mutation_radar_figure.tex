% Mutation-adequacy radar.
%
% Design rationale: a tabular layout displays per-cell deltas one
% number at a time and reads as six independent comparisons, which
% under-sells \biotest's distinguishing property -- a single
% framework spanning all six (SUT, format) cells. The radar makes
% that property visual: \biotest{} is a closed hexagon, and each
% coverage-guided baseline is a single point on its single
% applicable axis. Per-cell scores are printed at every vertex so
% no rigour is lost relative to the prior table.
%
% Visual layers, back to front:
%   1. concentric percentage rings (25/50/75/100)
%   2. six radial spokes (one per axis)
%   3. Pure-Random polygon (grey, dashed)        -- floor reference
%   4. \biotest{} polygon (blue, semi-transparent fill)
%   5. strongest-baseline markers (orange dots)  -- one per axis
%   6. star marker on the seqan3/SAM \biotest{} vertex
%      (the only axis where \biotest{} exceeds every baseline)
%   7. per-axis labels at the perimeter (compact, two lines)
%   8. inline legend at the upper-left of the radar bounding box
%
% Unit convention: \R is a cm-unitless number. TikZ polar
% coordinates of the form (angle:{\R*frac}) resolve to cm by
% default, which gives the intended radius in cm.

\begin{figure*}[!t]

\caption{Mutation kill rate (\%) per (SUT, format) cell.
\biotest{}'s closed polygon spans all six cells (four host
languages); each coverage-guided fuzzer applies to a single host
language and appears as a single point on its single applicable
axis. \biotest{} \emph{exceeds} every baseline on seqan3/SAM by
$+6.87$\,pp ($\bigstar$) and \emph{trails the strongest baseline}
by $1.5$ to $4.2$\,pp on the four close-margin pairs (htsjdk/VCF,
htsjdk/SAM, vcfpy/VCF, noodles/VCF) which all clear family-wise
$\alpha{=}0.05$ under Holm--Bonferroni
(\S\ref{sec:mutation-adequacy}). The only large gap is biopython/SAM
($-32.5$\,pp vs Atheris) where the parser is byte-content-lenient
and coverage-guided fuzzing is structurally
favoured~\citep{klees2018fuzz,boehme2020boosting}. Per-cell
$n_{\mathrm{BT}}/n_{\mathrm{base}}$, std (\biotest/baseline), and
mutation engine are listed in
\S\ref{sec:mutation-adequacy}.}
\label{fig:mutation-radar}
